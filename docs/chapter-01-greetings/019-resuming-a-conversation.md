# Pattern Family 19 — Resuming a Conversation

## Hindi Pattern

- तो मैं कह रहा था...
- जहाँ थे वहीं से...
- हाँ, तो...
- वैसे...

## Natural English Expressions

- Anyway...
- As I was saying...
- So, where were we?
- Getting back to...
- To continue...

## Nuance

These help you return to the original topic after an interruption.

## Examples

- Anyway, as I was saying...
- So, where were we?
- Getting back to the report...

## Common Mistakes

❌ As I was telling.

✅ As I was saying.

## Memory Tip

> "As I was saying..." is the classic way to resume.
