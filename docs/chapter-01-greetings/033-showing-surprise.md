# Pattern Family 33 — Showing Surprise

## Hindi Pattern

- अरे वाह!
- सच में!
- क्या बात है!

## Natural English Expressions

- Wow!
- Really?
- That's amazing!
- No way!
- You've got to be kidding!

## Nuance

Different expressions fit different levels of surprise.

## Examples

- Wow!
- Really?
- That's amazing!

## Common Mistakes

❌ What a thing!

✅ Wow!

## Memory Tip

> Use short reactions.
