# Pattern Family 27 — Checking If Someone Is Still There

## Hindi Pattern

- हेलो?
- सुन रहे हो?
- लाइन पर हो?

## Natural English Expressions

- Are you still there?
- Can you still hear me?
- Hello?
- Are you with me?

## Nuance

Useful during phone and video calls.

## Examples

- Are you still there?
- Can you still hear me?
- Hello?

## Common Mistakes

❌ Are you there only?

✅ Are you still there?

## Memory Tip

> Ask about presence, not 'only'.
