# Pattern Family 59 — Checking If Someone Has Time

## Hindi Pattern

- अभी समय है?
- दो मिनट मिलेंगे?
- अभी बात कर सकते हैं?

## Natural English Expressions

- Do you have a few minutes?
- Is now a good time?
- Can I steal a minute?
- Do you have some time?
- Can we chat for a bit?

## Nuance

Polite way to request someone's time.

## Examples

- Do you have a few minutes?
- Is now a good time?
- Can I steal a minute?

## Common Mistakes

Avoid literal translations; use the established English expressions.

## Memory Tip

> Learn the expression as a complete chunk.
