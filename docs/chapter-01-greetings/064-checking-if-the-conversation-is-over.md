# Pattern Family 64 — Checking Whether to End

## Hindi Pattern

- और कुछ?
- बस इतना ही?
- कुछ और था?

## Natural English Expressions

- Anything else?
- Is there anything else?
- Was that everything?
- Anything more?
- That's all, right?

## Nuance

Useful before ending a conversation.

## Examples

- Anything else?
- Is there anything else?
- Was that everything?

## Common Mistakes

Avoid literal translations.

## Memory Tip

> Learn the whole phrase, not individual words.
