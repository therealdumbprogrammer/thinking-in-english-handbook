# Pattern Family 66 — Checking If Someone Is Busy

## Hindi Pattern

- बिज़ी हो?
- व्यस्त तो नहीं हो?
- अभी बात कर सकता हूँ?

## Natural English Expressions

- Are you busy?
- Do you have a moment?
- Is this a bad time?
- Can I interrupt for a second?
- Got a minute?

## Nuance

Polite ways to check availability.

## Examples

- Are you busy?
- Do you have a moment?
- Is this a bad time?

## Common Mistakes

Avoid literal translations.

## Memory Tip

> Use these as fixed conversational phrases.
