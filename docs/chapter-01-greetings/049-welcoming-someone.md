# Pattern Family 49 — Welcoming Someone

## Hindi Pattern

- आइए।
- स्वागत है।
- अंदर आइए।

## Natural English Expressions

- Welcome!
- Come on in.
- Please come in.
- It's good to have you here.
- Make yourself at home.

## Nuance

Choose based on formality and familiarity.

## Examples

- Welcome!
- Come on in.
- Please come in.

## Common Mistakes

Avoid literal translations; use the standard expressions above.

## Memory Tip

> Learn these as fixed spoken phrases.
