# Pattern Family 47 — Checking If Everything Is Okay

## Hindi Pattern

- सब ठीक है?
- क्या हुआ?
- सब ठीक तो है?

## Natural English Expressions

- Is everything okay?
- Is everything alright?
- What's wrong?
- Is something bothering you?
- You okay?

## Nuance

Choose softer forms unless something is clearly wrong.

## Examples

- Is everything okay?
- Is everything alright?
- What's wrong?

## Common Mistakes

Avoid literal translations; use the standard expressions above.

## Memory Tip

> Learn these as fixed spoken phrases.
