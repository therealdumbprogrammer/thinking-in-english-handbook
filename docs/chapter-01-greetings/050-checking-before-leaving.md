# Pattern Family 50 — Before Leaving

## Hindi Pattern

- चलूँ?
- मैं निकलूँ?
- अब चलता हूँ?

## Natural English Expressions

- I'd better head out.
- I should get going.
- I'll get going now.
- Time for me to go.
- I should be on my way.

## Nuance

English often softens departures rather than asking for permission.

## Examples

- I'd better head out.
- I should get going.
- I'll get going now.

## Common Mistakes

Avoid literal translations; use the standard expressions above.

## Memory Tip

> Learn these as fixed spoken phrases.
