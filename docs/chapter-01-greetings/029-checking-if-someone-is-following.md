# Pattern Family 29 — Checking If Someone Is Following

## Hindi Pattern

- समझ आ रहा है?
- मेरी बात समझ रहे हो?
- फॉलो कर रहे हो?

## Natural English Expressions

- Are you following?
- Does that make sense?
- Are you with me?
- Following so far?

## Nuance

Often used while explaining something.

## Examples

- Are you following?
- Does that make sense?
- Are you with me?

## Common Mistakes

❌ Are you understanding?

✅ Are you following?

## Memory Tip

> Prefer 'following' while explaining.
