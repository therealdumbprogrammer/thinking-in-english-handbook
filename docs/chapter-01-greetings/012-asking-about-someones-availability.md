# Pattern Family 12 — Asking About Someone's Availability

## Hindi Pattern

- फ्री हो?
- बिज़ी तो नहीं हो?
- दो मिनट हैं?
- टाइम है?
- अभी बात कर सकते हो?

## Natural English Expressions

| Expression | Register |
|---|---|
| Do you have a minute? | Neutral |
| Got a minute? | Casual |
| Are you busy right now? | Neutral |
| Is now a good time? | Neutral |
| Can we talk for a minute? | Neutral |

## Nuance

These expressions politely check whether someone is available before starting a conversation.

## Examples

- Do you have a minute?
- Got a minute?
- Is now a good time to talk?
- Are you busy right now?
- Can we talk for a minute?

## Common Mistakes

❌ Are you free?

✅ Do you have a minute?

**Why?** Native speakers often ask about time rather than "freedom."

## Similar English Expressions

- Do you have a few minutes?
- Mind if I ask you something?

## Memory Tip

> Ask for a minute, not whether someone is free.

## Mini Practice

Hindi → English: बिज़ी तो नहीं हो?

English → Hindi: Is now a good time?

## Quick Recap

Most common: Do you have a minute?
