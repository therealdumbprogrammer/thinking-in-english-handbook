# Pattern Family 36 — Politely Disagreeing

## Hindi Pattern

- ऐसा नहीं है।
- मुझे नहीं लगता।
- मैं सहमत नहीं हूँ।

## Natural English Expressions

- I'm not so sure about that.
- I don't think so.
- I see it differently.
- I'm afraid I disagree.
- Not necessarily.

## Nuance

Disagree without sounding confrontational.

## Examples

- I'm not so sure about that.
- I don't think so.
- I see it differently.

## Common Mistakes

❌ You are wrong.

✅ I'm not so sure about that.

## Memory Tip

> Disagree with the idea, not the person.
