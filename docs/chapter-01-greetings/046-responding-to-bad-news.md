# Pattern Family 46 — Responding to Bad News

## Hindi Pattern

- अरे यार...
- ओह नहीं!
- ये तो बुरा हुआ।

## Natural English Expressions

- I'm sorry to hear that.
- Oh no!
- That's unfortunate.
- That's too bad.
- I hope things get better.

## Nuance

Express empathy before asking questions.

## Examples

- I'm sorry to hear that.
- Oh no!
- That's unfortunate.

## Common Mistakes

Avoid literal translations; use the standard expressions above.

## Memory Tip

> Learn these as fixed spoken phrases.
